Tic-Tac-Toe

2 player Tic-Tac-Toe game created using Python. Player 1 is Xs and Player 2 is Os. The main board is displayed using a hash table with 9 key values. Players pick their board locations during their turn by using the integers 1-9 after the board is displayed to them. After every turn a win condition function checks if there is a winner. If there is a winner the game stops and prints the board and displays who won the game. If the global turn counter variable gets to a value of 9 (max number of turns) and there is still no winner the game ends in a tie.

Rock-Paper-Scissors

1 player rock paper scissors game created using Python. Player picks either 'r' for rock, 'p' for paper, or 's' for scissors, or even 'q' to quit. Using a hash table the player's 'r', 'p', and 's' will be converted to a 1, 2, or 3. The computer will randomly pick a choice 1-3 and store the result. The hash table will then determine if the player won, lost, or tied depending on what the player chose versus what the computer chose. The result along with a running score of both the computer will be displayed. The player can quit after any round they want using 'q' and the final score will be displayed before exiting the program.

  • Github Repository: Rock-Paper-Scissors

Facebook-Spambot

The program will use a person's Facebook email and password as input to login to Facebook and go to the Facebook messenger utilizing the Selenium WebDriver library to get a page's HTML. The program will then use a friend's unique Facebook ID to send a "This is a bot" message every 10 seconds until the program is stopped.
